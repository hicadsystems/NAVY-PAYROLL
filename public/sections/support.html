System Flow Guide

1. How to Add New Entry

Click the "+ Add New" button → Fill in required fields → Submit

2. How to Generate Reports

Click "Generate Report" button → Select date range → Choose format → Download

3. How to Edit Records

Find record in table → Click "Edit" → Modify fields → Save changes

4. How to Delete Records

Find record in table → Click "Delete" → Confirm deletion

5. Navigation

Use sidebar menu to access different sections → Dashboard, Administration, Payroll, etc.

6. File Updates

Go to File Update section → Upload file → Verify data → Process update
